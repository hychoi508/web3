web page 이동 개발 방식

dd tag 소개 url

1. html tag중 링크 tag를 이용해서 daum 사이트로 이동될 수 있도록 완성하기 : 링크는 무조건 get
다음 사이트로 이동

2. html button tag를 click해서 daum 사이트로 이동 : get방식


3. servlet api로 forword[디스페처] 방식으로 화면 이동하기
3-1. 실행되는 file 순서 : index.html -> Page1Servlet -> Page2Servlet

3-2. file간 이동 방식 : index.html: 링크 -> Page1Servlet : forword -> Page2Servlet

3-3. client 브라우저에 출력하는 로직은 Page1Servlet에서 개발하지 않음
3-4. Page1Servlet 에선 요청 객체에 name이라는 key로 개인 이름 저장해서 Page2Servlet에서 출력하기
3-5. 클릭하기 : Page1Servlet 호출하기


4. servlet api로 redirect 방식으로 화면 이동하기
4-1. 실행되는 file 순서 : index.html -> Page3Servlet -> Page4Servlet

4-2. file간 이동 방식 : index.html: form tag -> Page3Servlet : redirect -> Page4Servlet

4-3. client 브라우저에 출력하는 로직은 Page3Servlet에서 개발하지 않음
